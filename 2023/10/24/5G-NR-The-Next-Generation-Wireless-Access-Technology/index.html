5G NR The Next Generation Wireless Access Technology

5G NR

  • FDD 频分 TDD 时分

    FDD: 基站给 ue 发送信息用 f1 的频率，ue 给基站发送信息使用 f2 的频率.

    TDD: 基站和 ue 之间通信使用相同的频率，但在不同的时间段上进行切换（基站发送给 ue 切换到 ue 发送给基站）.

  • Time-Domain Structure

    Transmission 被分为长度为 10 ms 的 frame，每一个 frame 被分为长度为 1 ms 的 subframe，每个 subframe 由包含 14 个 OFDM symbols 的 slot 组成.

    每个 frame 有一个唯一的 System Frame Number (SFN)，SFN 的周期为 1024，SFN 会在1024 个 frame，或者说是 10.24 s 后出现.


  • Physical-Layer Control Signaling

    • Downlink

      Physical Layer - L1

      Mac Layer - L2

      • Downlink 在干什么
        1. Downlink scheduling assignments (UE 能够正常接受，解调和解码 DL-SCH 所需要的信息)
        2. Uplink scheduling grants (通知 UE 关于 UL-SCH 的传输资源和传输格式)
        3. Special purpose (symbols used for uplink and downlink in a set of slots, preemption indication, and power control)

      NR 中，只有一个 signal control channel type - Physical downlink control channel (PDCCH).

      • Physical Downlink Control Channel (PDCCH)


      payload - Downlink Control Information (DCI)

      24-bit CRC is attached to detect transmission errors.

      RNTI modifies the CRC by using a scrambling operation.

      DCI 通过解码 CRC 得出，并不会 explicitly 传输，从而减少 bits 的使用.

      The RNTI does NOT necessarily have to be the identity of the device, the C-RNTI, but can also be different types of group or common RNTIs, for example to indicate paging or a random-access response.

      • Control-channel Elements (CCEs) - 当系统给 UE 发送控制信息的时候，需要使用 PDCCH。而 CCEs 是 PDCCH 使用的资源的基本单位，它指定了哪些资源用于 PDCCH 传输.
      • Resource-element groups (REGs) - 一组 Resource element. RE 可以看作是二维时间-频率网格上的一个点.

      PDCCH的资源是按REGs来分配的，而不是单独的RE.

      • CORESET

        CORESET is a set of physical resources(i.e, a specific area on NR Downlink Resource Grid) and a set of parameters that is used to carry PDCCH/DCI.

        和 LTE 不同的是，NR CORESET 是在频域中的一个特定的区域.


        A CORESET can accommodate multiples of Aggregation Level. An Aggregation Level is made up of N CCE. A CCE is made up of 6 REGs. A REG is made up of 1 RB and 1 OFDM Symbol.


      • RE, REG, REG Bundles, CCE, Aggregation Level, CORESET 的组成


        RE: the smallest unit of the resource grid.

        REG: it is made up of 12 resource elements in frequency domain and one OFDM symbol in time domain.

        REG Bundles: multiple REGs. The size of bundles is L which is determined by the RRC parameter reg-bundle-size .

        CCE: multiple REGs.

        Aggregation Level: indicates how many CCEs are allocated.

        CORESET: CORESET is defined from a device perspective and only indicates where a device may receive PDCCH transmissions. It does not say anything on whether the gNB actually transmits a PDCCH or not.

  • K0, K1, K2的含义

    5G-NR K0, K1 & K2

Random Access Channel (RACH)

  • Aim: For establishing Uplink Synchronization between a device and a network (gNB).
  • 手机/电脑第一次连接网络的时候（或者很长时间之后再次连接网络的时候），需要 RACH 去请求访问网络.

RACH Overall Procedure


Contention Based RACH Procedure

在 UE 以一种特定的模式发送 Preamble，被称为 Signature. UE 随机选择一个 Signature，因此会出现冲突，即多个 UE 选择了完全相同的 Signature.

会通过 “Contention Resolution” step 来解决冲突.

  • 步骤：
    • UE → gNB: Preamble (RA-RNTI, L2/L3 消息大小)
    • UE ← gNB: TC-RNTI
    • UE → gNB: L2/L3 msg

如果多个 UE 选择了完全相同的 Preamble，gNB 只会解析其中一个的信息，其余的 UE 就会知道发生了冲突，从而回到第一步重新进行连接的步骤.

成功的 UE 会收到一个唯一的 C-RNTI.

Non-Contention Based RACH Procedure

步骤：

  • UE ← gNB: PRACH
  • UE → gNB: RA-RNTI, L2/L3 消息大小
  • UE ← gNB: C-RNTI

gNB 向特定的 UE 提供 random access resources.

This method is typically used in scenarios where the network has prior knowledge about the UE’s intention to initiate communication. For example, it can be used for handover procedures or when the network anticipates uplink data transmission from the UE based on downlink data or signaling.

Preamble Structure

Basic Preamble

由长度为 L 的 preamble 序列 $p_0,p_1,\dots,p_{L-1}$ 生成，基于 Zadoff-Chu 序列. 也可以有由相同根序列的不同循环位移生成.

Different preamble sequences can be generated from different Zadoff- Chu sequences corresponding to different root-sequence indices. However, it can also be generated from different cyclic shifts of the same root sequence.

  • 循环位移 Cyclic Shift

    假设有一个基于 Zadoff-Chu 序列的根序列 $R = [r0, r1, r2, r3]$，通过循环移位，可以生成以下序列：

    • 循环移位0次：$[r0, r1, r2, r3]$
    • 循环移位1次：$[r3, r0, r1, r2]$
    • 循环移位2次：$[r2, r3, r0, r1]$
    • 循环移位3次：$[r1, r2, r3, r0]$

    这些生成的诗序可以保证彼此的正交，进而允许多个用户同时尝试接入网络而互不干扰.

The set of cyclic shifts 由 zero-correlation-zone 参数给出.

zero-correlation-zone = orthogonality.

在一个 cell 中，最多由 64 种不同的 preamble，其对应的索引是 0 ~ 63.
