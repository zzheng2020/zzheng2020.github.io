字符串哈希

字符串哈希（前缀哈希）
by ZhangZiheng

题目描述：
给定一个长度为n的字符串，再给定​m​个询问，每个询问包含四个整数$ l_1,r_1,l_2,r_2 $，请你判断$[l_1,r_1]$和$[l_2,r_2]$这两个区间所包含的字符串子串是否完全相同。
字符串中只包含大小写英文字母和数字。

输入格式：
第一行包含整数n和m，表示字符串长度和询问次数。
第二行包含一个长度为n的字符串，字符串中只包含大小写英文字母和数字。
接下来m行，每行包含四个整数$l_1,r_1,l_2,r_2$，表示一次询问所涉及的两个区间。
注意，字符串的位置从1开始编号。

输出格式：
对于每个询问输出一个结果，如果两个字符串子串完全相同则输出“Yes”，否则输出“No”。
每个结果占一行。

数据范围：
$1\leq n \leq m \leq 10^5$

样例：

// input
8 3
aabbaabb
1 3 5 7
1 3 6 8
1 2 1 2

// output
Yes
No
Yes

  • 在需要多次询问的题目中，每次对字符串进行哈希会导致效率过低，因此可以预处理出哈希结果。
/*
h[]: hash结果
p[]: p[i]表示p的i次幂
str[]: 需要哈希的字符串

const int P = 13331;
unsigned long long h[N], p[N];
*/
for (int i = 1; i <= n; i++)
{
    p[i] = p[i - 1] * P;
    h[i] = h[i - 1] * P + str[i];
}

避免将字符映射成0，如若不然，类似于a，aa的字符串将有相同的哈希值，将无法用哈希结果判断字符串是否相同。

  • 当有多组询问时，只需考虑$hash(l_1, r_1)$与$hash(l_2,r_2)$的值是否相等，即可判断两个子串是否相同。

具体代码实现如下：时间复杂度$O(n)$

#include <stdio.h>
#include <iostream>

using namespace std;
/*
h[]: hash结果
p[]: p[i]表示p的i次幂
str[]: 需要哈希的字符串
*/
const int N = 1e5 + 10, P = 13331;

int n, m;
unsigned long long h[N], p[N];
char str[N];

// 获取[l,r]的哈希值
unsigned long long get(int l, int r)
{
    return h[r] - h[l - 1] * p[r - l + 1];
}

int main()
{
    cin >> n >> m >> (str + 1);
    p[0] = 1;
    for (int i = 1; i <= n; i++)
    {
        p[i] = p[i - 1] * P;
        h[i] = h[i - 1] * P + str[i];
    }

    while(m--)
    {
        int l1, r1, l2, r2;
        cin >> l1 >> r1 >> l2 >> r2;
        cout << (get(l1, r1) == get(l2, r2) ? "Yes" : "No") << endl;
    }
    return 0;
}
