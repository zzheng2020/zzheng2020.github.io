线段树(带懒惰标记的更新)

线段树(带懒惰标记的更新)
by ZhangZiheng

(填坑ing)

在上一篇关于线段树的文章中 ，叙述了单点更新、查找的相关内容。接下来，将叙述线段树带懒惰标记的相关操作。

为什么要带lazy标记（为了躺平）

在修改操作中，可能会对区间进行修改，如果每次都直接找到叶子节点进行修改的话，那么一次修改操作的最坏时间复杂度将达到$O(n)$，显然对于修改操作来说，这个时间复杂度是无法接受的。（愉快地TLE了）

因此，引入了懒惰标记和pushdown操作来解决上述问题。

懒惰标记的含义

在维护节点信息时，额外维护一个add信息，表示“当前节点已经被修改，但其覆盖的其他子节点未被修改”。这样一来，就不必每次修改时都找到叶子节点进行修改，从而使得时间复杂度降低，为$O(log\ n)$。

添加懒惰标记的注意事项

当从上往下递归的时候，为了保证查询结果正确，如果该节点的祖宗节点仍然存在懒惰标记，需要将其清空，并把懒惰标记向两个儿子节点传递。换句话说，查询到某个节点需要返回时，这个节点的祖宗节点都没有了懒惰标记（因为懒惰标记已经向下传递了）。

实现上述过程其实非常简单，在每次查询过程中加入pushdown操作即可。

具体来说，pushdown操作如下：（以维护区间和的问题为例）

// 父节点的lazy标记传递给左儿子
left.lazy_add += root.lazy_add;
left.sum += (left.r - left.l + 1) * root.lazy_add;

// 父节点的lazy标记传递给右儿子
right.lazy_add += root.lazy_add;
right.sum += (right.r - right.r + 1) * root.laze_add;

// 注意: 不要忘了清空祖宗节点的lazy标记
root.lazy_add = 0;

例题：一个简单的整数问题

题目描述

给定一个长度为$N$的数列$A$，以及$M$条指令，每条指令可能是以下两种之一：

  1. “C l r d”，表示把$A[l],A[l+1],\dots,A[r]$ 都加上$d$。

  2. “Q l r”，表示询问 数列中第$l\sim r$个数的和。

对于每个询问，输出一个整数表示答案。

输入格式

第一行两个整数$N,M$。

第二行$N$个整数$A[i]$。

接下来$M$行表示$M$条指令，每条指令的格式如题目描述所示。

输出格式

对于每个询问，输出一个整数表示答案。

每个答案占一行。

数据范围

$1\leq N,M\leq 10^5, |d|\leq 10000, |A[i]|\leq 1000000000$

代码实现

#include <bits/stdc++.h>

using namespace std;

const int N = 1e5 + 10;

int n, m;
int a[N];
struct Node
{
    int l, r;
    long long sum, add;
}tr[N * 4];

void pushup(int u)
{
    tr[u].sum = tr[u << 1].sum + tr[u << 1 | 1].sum;
}

void pushdown(int u)
{
    auto &root = tr[u], &left = tr[u << 1], &right = tr[u << 1 | 1];
    if (root.add)
    {
        left.add  += root.add, left.sum += (long long)(left.r - left.l + 1) * root.add;
        right.add += root.add, right.sum += (long long)(right.r - right.l + 1) * root.add;
        root.add = 0;
    }
}

void build(int u, int l, int r)
{
    if (l == r) tr[u] = {l, r, a[l], 0};
    else
    {
        tr[u] = {l, r};
        int mid = (l + r) >> 1;
        build(u << 1, l, mid), build(u << 1 | 1, mid + 1, r);
        pushup(u);
    }
}

void modify(int u, int l, int r, int c)
{
    if (l <= tr[u].l && tr[u].r <= r)
    {
        tr[u].sum += (long long)(tr[u].r - tr[u].l + 1) * c;
        tr[u].add += c;
    }
    else
    {
        pushdown(u);
        int mid = (tr[u].l + tr[u].r) >> 1;
        if (l <= mid) modify(u << 1, l, r, c);
        if (r > mid) modify(u << 1 | 1, l, r, c);
        pushup(u);
    }
}

long long query(int u, int l, int r)
{
    if (l <= tr[u].l && tr[u].r <= r) return tr[u].sum;
    else
    {
        pushdown(u);
        int mid = (tr[u].l + tr[u].r) >> 1;
        long long sum = 0;
        if (l <= mid) sum += query(u << 1, l, r);
        if (r > mid) sum += query(u << 1 | 1, l, r);
        return sum;
    }
}

int main()
{
    scanf("%d%d", &n, &m);
    for (int i = 1; i <= n; i++) scanf("%d", &a[i]);
    build(1, 1, n);
    while (m--)
    {
        int l, r, c;
        char ch[2];
        scanf("%s", ch);
        scanf("%d%d", &l, &r);
        if (*ch == 'C')
        {
            scanf("%d", &c);
            modify(1, l, r, c);
        }
        else printf("%lld\n", query(1, l, r));
    }
    return 0;
}
